# Reuters

## What It Is
Reuters news browsing notes for current headlines and finance items.

## When To Use
- Latest news
- Market-moving headlines
- Finance and macro event scanning

## Recommended Flow
1. Open Reuters or reuse an existing Reuters tab.
2. Read the current top headlines.
3. Capture headline text and links before doing deeper analysis.
4. Keep market-impact analysis as a separate reasoning step.

## Notes
- Prefer headline extraction over deep parsing in the first pass.
- If navigation is ambiguous, select the target tab before issuing `goto`.
