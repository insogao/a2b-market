# Baidu Search

## What It Is
Baidu search workflow notes for Chinese-language query tasks.

## When To Use
- Searching Chinese queries
- Tasks that need Baidu-specific DOM behavior

## Recommended Flow
1. Open or select the Baidu tab.
2. Enter the query.
3. If Enter does not submit the query, click the visible search button.
4. Verify the results page loaded before extracting data.

## Notes
- Keep the extraction task separate from the search-submission task.
- Prefer A2B high-level commands before `run-js`.
