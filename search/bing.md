# Bing Search

## What It Is
Bing search workflow notes for web search and image search tasks.

## When To Use
- Search tasks where Bing is the target engine
- Image search tasks that need Bing-specific results pages

## Recommended Flow
1. Reuse an existing Bing tab when possible.
2. Submit the query with high-level commands first.
3. Confirm the results page is stable before scraping.

## Notes
- Prefer a visible submit action if keypress-based submission is unreliable.
