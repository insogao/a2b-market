# Google Search

## What It Is
Google search workflow notes for general web search tasks.

## When To Use
- General search tasks
- Image search or research tasks that fit Google better

## Recommended Flow
1. Reuse an existing Google tab if one already exists.
2. Navigate to the target query.
3. Confirm the results page has finished loading before extracting text or images.

## Notes
- Keep this entry focused on search behavior, not general browser automation.
- Prefer selecting the target tab before navigation when there is any ambiguity.
